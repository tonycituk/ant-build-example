## Java project example using ant for tasks.

### To clean
```bash
ant clean
```

### To compile
```bash
ant compile
```

### To package
```bash
ant jar
```

### To run
```bash
ant run
```

### Important!

In order to us this commands yo should:

- Install [ant](https://ant.apache.org/).
- Add & modify [this file](./build.xml) to your needs.
